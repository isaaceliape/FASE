# FASE - Framework de Automação Sem Enrolação

Bem-vindo à documentação do **FASE**, a versão em português brasileiro do **Get Shit Done (GSD)** — um sistema de meta-prompting e context engineering que faz os assistentes de IA funcionarem de verdade em projetos de software.

## 🚀 Comece Rápido

### Instalação

```bash
npx fase-ai
```

Escolha seu runtime preferido (Claude Code, OpenCode, Gemini, Codex, GitHub Copilot ou Qwen Code) e comece a usar.

### Seu Primeiro Projeto

```bash
/fase-novo-projeto
/fase-planejar-fase
/fase-executar-fase
```

## ✨ Características Principais

- **🤖 Agents Paralelos** - Múltiplos agentes de IA trabalhando simultaneamente
- **🎯 Spec-Driven** - Desenvolvimento guiado por especificação
- **🏗️ Arquitetura Limpa** - Separação clara de responsabilidades
- **🌐 Multi-Runtime** - Funciona com 6 assistentes de IA diferentes
- **📝 Context Aware** - Gerenciamento inteligente de contexto
- **✅ Validação Automática** - Verificação de fase automática

## 📊 By The Numbers

- **13** Agentes especializados
- **34** Comandos em português
- **6** Runtimes suportados (Claude Code, OpenCode, Gemini, Codex, GitHub Copilot, Qwen Code)
- **100%** Spec-driven development
